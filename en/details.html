Using Bluemix Kibana dashboards to view IBM Integration Bus Logs

Learn how to use a Kibana Dashboard on Bluemix to get a centralized view of IBM Integration Bus Logs.
OVERVIEW

IBM Integration Bus provides a facility to send logging information from configured integration servers to IBM Bluemix where you can display the reported information in a Kibana dashboard. This logging service is currently provided free of charge but is not yet recommended for use with a production environment (because IBM has not yet provided guarantees regarding large volumes of data and log retention periods).

This tutorial demonstrates how IBM Integration Bus provides a sample (version 4) Kibana dashboard. Graphical and tabular views enable you to see important information quickly and easily. Users can change or replace these views with charts and tables of their own.

This tutorial takes approximately 10 minutes.
