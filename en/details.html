Hybrid Connect Logging

Learn how to register IBM Integration Bus with the Hybrid Connect logging (Logmet) service.
OVERVIEW

This tutorial demonstrates how to display the IBM Integration Bus Dashboard using Kibana 4 in the Hybrid Connect Logging (Logmet) service.

The Logmet service can be set up to receive logs from IBM Integration Bus. IBM Integration Bus provide a sample Kibana dashboard that user can change or replace with charts and tables of their own. You can use this to do data discovery and data analysis.

This tutorial takes approximately 10 minutes.
