Import projects

When you click Import, the LogMessages application will be created in you workspace.

LogMessages uses the Compute node to log different types of messages into the system log and also a Trace node for tracking the execution of the flow. This will be use as a way to create data for the Logmet service.

Configuring IBM Integration Bus Kibana dashboard

Follow the instructions below to configuration an integration server to the IBM Integration Bus Kibana dashboard.

  1. Follow the instructions in Configuring the integration server and loading the IBM Integration Bus Kibana dashboard manually of the Knowledge Center to setup and import all thes into Kibana 4.
  2. Once everything are imported, select the Dashboard tab under Kibana 4 and click on Load Saved Dasboard (open folder) button.
  3. Under the Dashboard Filter entry field, select IBM Integration Bus Dashboard, this will open the imported dashboard with the predefined visuals.
  4. If the visuals are displaying the message No results found, try changing the time filter (clock icon on top right) to a larger value.

Deploy the LogMessages application to your integration server and run it a few times to generate some data for the log.

Use the instruction below to create a custom chart to track a specific IBM Integration Bus log messages:

  1. In Kibana 4 click on Visualize and then Vertical bar chart.
  2. Select From a saved search, and then IBM Log Events.
  3. Click on X-Axis as the bucket type.
  4. Under Aggregation, select Date Histogram.
  5. Click on Add sub-bucketsSplit Bars.
  6. In the Sub Aggregation field select Filters.
  7. Enter ibm_messageId:3051 into Filter 1 and press the Apply changes (white triangle in a green background) button to update the chart to show data for message id 3051.
  8. Click on the Save Visualize (small disk icon on top right) to save the chart.
  9. Once the chart is saved, it can be add to the dashboard by clicking on the Add Visualization button.
  10. Select the new chart from the list and it will be added to the bottom of the dashboard. You can move and resize it, or replace an existing one in the dashboard.

This tutorial has shown how to register IBM Integration Bus with Hybrid Connect Logging service, and create new charts in Kibana to track a particular message.
